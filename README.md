# 🧮 Maths Game

A browser-based multiplication quiz game built using **HTML**, **CSS**, and **Vanilla JavaScript**. The game challenges players to solve random multiplication problems within a limited time while keeping track of their score.

## 🚀 Features

- 🎯 Random multiplication questions
- ⏱️ 60-second countdown timer
- 📊 Live score tracking
- ✅ Instant feedback for correct answers
- ❌ "Try Again" feedback for incorrect answers
- 🔄 Start and Reset game functionality
- 🎲 Randomized answer placement
- 🏁 Game Over screen displaying the final score

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
